Algorithm Efficiency

Algorithm efficiency is the study of the amount of resources used by an algorithm. The less resources used by the algorithm the more efficient it is.

  • Time efficiency :
    Amount of time an algorithm takes to solve a problem.

  • Space efficiency :
    Amount of memory an algorithm needs to solve a problem.

  • Complexity theory :
    The performance of an Algorithm based on cost functions of statement counts.


Nevertheless, to have a good comparison between different algorithms we can compare based on the resources it uses :

  • how much time it needs to complete
  • how much memory it uses to solve a problem
  • how many operations it must do in order to solve the problem
